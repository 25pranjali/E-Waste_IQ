# 🌿 E-Waste IQ PWA

**Think Smart. Recycle E-Waste.**

E-Waste IQ is a modern, responsive Progressive Web Application (PWA) designed to educate the public about electronic waste (e-waste) and promote responsible recycling habits. Built with a focus on **Green IT principles**, the application features a minimalist, energy-efficient design and offline capabilities.

## 🌟 Key Features

- **Interactive Map**: Functional recycling center locator integrated with Leaflet.
- **Educational Guide**: Deep dives into e-waste types, environmental impact, and India's E-Waste Rules 2022.
- **Gamified Quiz**: Test your knowledge and earn "Eco Warrior" badges.
- **Green UI**: A premium, high-contrast design with a dedicated dark mode for energy saving.
- **Offline Support**: Full PWA integration allows access to educational content even without an internet connection.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **PWA**: [vite-plugin-pwa](https://vite-pwa-org.netlify.app/)
- **Maps**: [Leaflet](https://leafletjs.com/) & [React-Leaflet](https://react-leaflet.js.org/)
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics.
- **Assets**: Custom 3D isometric illustrations generated for a premium look.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (v9 or higher)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/25pranjali/E-Waste_IQ.git
   cd E-Waste_IQ
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🌍 Green IT Principles

This project was built with sustainability in mind:
- **Energy-Efficiency**: Minimalist UI and dark mode to reduce device power consumption.
- **Optimized Performance**: Lightweight code and assets for fast loading and low carbon footprint.
- **Offline Accessibility**: Reducing server requests through aggressive caching.
